Social Media Application - Scrolint

It is a MERN stack application where people can connect to each other. People can post their pictures or comments. They can even like and comments on their friends posts as well.

Technologies used :

FRONT-END:
Framework        - ReactJs
Navigation       - React router
Form validation  - FORMIK + yup
State management - Redux toolkit
local storage    - Redux persistant storage
Image uploads    - React dropzone


BACK-END: 
Backend framework    - express Js
process request body - body-parser
passoword encryption - bcrypt 
cross origin request - cors
environment variables- dotenv 
file upload          - gridfs-stream
upload file locally  - multer and multer-gridfs-storage
request safety       - helmet
for logging          - morgan
Authentication       - jsonwebtoken
Database             - mongoose (MongoDB)
